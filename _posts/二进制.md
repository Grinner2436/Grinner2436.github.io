# 二进制

### Reids命令
```
参考书：
  《Reids设计与实现3.0》22章
  《Redis开发与运维》3章5节

1.设置某位置bit值，从右往左，首索引0，
IN: SETBIT key_name position_from_right value_1_or_0
RE: 指定偏移量原来储存的位, 异常情况返回0
> SETBIT bitkey 0 1
0

2.取某位置bit值，从右往左，首索引0，
IN: GETBIT key_name position_from_right
RE: value_1_or_0, 异常情况返回0
> GETBIT bitkey 0
1

3.统计数组内1的数量
IN: BITCOUNT key_name
RE: count
> BITCOUNT bitkey
1
#Redis使用SWAR算法降低映射表键长，然后使用查表法查1字节（8位）的键得出长度。

4.位运算
IN: BITOP operation_name result_set_key_name op_key1 op_key2 op_key3...
RE:
operation_name :
  与（&）：AND
  或（|）：OR
  非（~）：NOT
  异或（^）：XOR
> BITOP NOT result bitkey
1
```
[Redis commands supported by RedisTemplate](https://docs.spring.io/spring-data/redis/docs/current/reference/html/#appendix:command-reference)
