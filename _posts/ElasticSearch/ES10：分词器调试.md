ES10：分词器调试

### 使用分词器查看分词结果
```
GET _analyze
{
  "tokenizer" : "keyword",
  "filter" : ["lowercase"],
  "char_filter" : ["html_strip"],
  "explain"  :  true,
  "text" : "this is a <b>test</b>"
}
```
